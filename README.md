# VietClaw landing

Landing page for VietClaw (Next.js 14, Tailwind CSS, shadcn/ui). Brand: dark navy `#0A0E2E`, electric cyan `#00D4FF`, Inter.

## Chạy local

```bash
npm install
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000).

```bash
npm run build   # production build
npm run lint    # ESLint
```
